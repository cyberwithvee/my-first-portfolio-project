## Cross-Platform Expense Tracker Ecosystem

A modern personal finance tool designed to showcase cross-platform feature alignment. This repository demonstrates standard development practices for both a web analytics dashboard and a mobile-first user interface.

---

## 📂 Project Structure

* **`web-dashboard/`** — Built as a full-viewport desktop layout for long-session financial analytics.
* **`mobile-interface/`** — Built with mobile-responsive viewports to mimic native application interfaces for rapid data entry.
* **`app.js`** — The shared JavaScript data engine using the Web Storage API (`localStorage`) to sync simulated data between both interfaces instantly.

---

## 🛠️ Tech Stack & Methods
* **Frontend:** Semantic HTML5, Modular CSS3
* **Logic Layer:** Modern JavaScript (ES6+)
* **Data Syncing:** Web Storage API (`localStorage`)

---

## 💻 How to Preview
1. Clone or download this repository to your machine.
2. Open `/mobile-interface/index.html` in a browser window to submit an expense.
3. Open `/web-dashboard/index.html` in a separate window to watch the analytics dashboard update dynamically!
